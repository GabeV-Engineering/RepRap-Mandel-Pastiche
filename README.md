BED:
The Bed consists of 2 lasercut aluminum sheets and a 150mm x 150mm heater.

HOTEND:
This machine uses an Azoto Protossido hotend (custom) it requires soldering together flatpacked lasercut copper.  An E3D version of the toolhead can be designed upon request.

FRAME:
The current files are using 9.4mm holes.  3/8" Rods are what was used in the final build.  The .step files can be modified to acomodate 8mm rods.

FIRMWARE:
This printer uses Marlin Firmware (GPL v3).
Configuration files for this machine are provided.
Download the latest Marlin release and replace the default configuration files with these versions.
